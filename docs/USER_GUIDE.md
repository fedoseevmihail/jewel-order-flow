## Общие сведения

**JewelCraft ERP** — веб-приложение для управления заказами на изготовление ювелирных изделий. Система позволяет клиентам создавать заказы с загрузкой 3D-моделей (STL-файлы), а сотрудникам производства — вести заказы по этапам изготовления.

### Технологический стек

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **3D-просмотр:** Three.js, React Three Fiber

---

## Роли пользователей

| Роль | Описание | Доступные действия |
|------|----------|--------------------|
| **Клиент** | Заказчик ювелирного изделия | Создание заказов, загрузка STL-файлов, просмотр своих заказов, отправка черновика в работу |
| **Оператор** | Специалист по выращиванию | Просмотр заказов на этапе «В росте», перевод заказа на следующий этап |
| **Литейщик** | Специалист по литью | Просмотр заказов на этапе «В литье», перевод заказа на следующий этап |
| **Администратор** | Управляющий системой | Полный доступ: управление пользователями, все заказы, любые действия |

## Регистрация и вход

### Регистрация клиента

1. Откройте приложение в браузере.
2. На странице входа перейдите на вкладку **«Регистрация»**.
3. Заполните поля:
   - **Имя** — ваше полное имя (ФИО).
   - **Email** — адрес электронной почты (будет использоваться для входа).
   - **Пароль** — минимум 6 символов.
4. Нажмите **«Зарегистрироваться»**.
5. Проверьте электронную почту и подтвердите регистрацию по ссылке в письме.
6. После подтверждения войдите с вашим email и паролем.

> **Примечание:** При регистрации автоматически назначается роль «Клиент». Другие роли назначает Администратор.

### Вход в систему

1. На странице входа введите **Email** и **Пароль**.
2. Нажмите **«Войти»**.
3. Вы будете перенаправлены на Дашборд.

### Выход

Нажмите иконку выхода (🚪) в правом верхнем углу шапки приложения.

## Клиент

### Дашборд

После входа клиент видит страницу **«Мои заказы»** со списком всех своих заказов. Заказы можно фильтровать по статусу с помощью вкладок:

- **Все** — все заказы
- **Черновик** — сохранённые, но не отправленные
- **Новый** — отправленные в работу
- **В росте**, **В литье**, **В отгрузке**, **Завершён** — заказы на различных этапах производства

### Создание заказа

1. Нажмите **«Новый заказ»** в навигации.
2. Заполните форму:
   - **Название изделия** (обязательно) — например, «Кольцо с бриллиантом».
   - **Описание** — подробные требования к изделию.
3. Загрузите **STL-файлы**:
   - Нажмите **«Выбрать файлы»** или перетащите файлы в область загрузки.
   - Поддерживаются файлы формата `.stl`.
   - Можно загрузить несколько файлов.
   - Для удаления файла из списка нажмите ✕ рядом с ним.
4. Выберите действие:
   - **«Сохранить черновик»** — заказ сохраняется со статусом «Черновик», можно вернуться и отправить позже.
   - **«Отправить заказ»** — заказ сразу получает статус «Новый» и попадает в очередь на обработку.

### Просмотр заказа

Нажмите на карточку заказа на Дашборде. На странице заказа отображаются:

- **Название** и **описание** заказа.
- **Статус** — текущий этап с прогресс-баром.
- **Дата создания** и **имя клиента**.
- **STL-файлы** — список загруженных файлов с возможностью скачать (⬇) и просмотреть в 3D (👁).

### Отправка черновика

Если заказ в статусе «Черновик», клиент может нажать кнопку **«Перевести: Новый»**, чтобы отправить заказ в работу.

## Оператор

### Дашборд

Оператор видит страницу **«Все заказы»** с пометкой «Заказы на этапе роста и далее». Отображаются все заказы системы с фильтрацией по статусу.

### Работа с заказами

1. Откройте заказ со статусом **«Новый»**.
2. Нажмите **«Перевести: В росте»** — заказ переходит на этап роста, а вы назначаетесь ответственным оператором.
3. После завершения роста нажмите **«Перевести: В литье»** для передачи заказа литейщику.

---

## Литейщик

### Дашборд

Литейщик видит страницу **«Все заказы»** с пометкой «Заказы на этапе литья и далее».

### Работа с заказами

1. Откройте заказ со статусом **«В литье»**.
2. После завершения литья нажмите **«Перевести: В отгрузке»** — заказ переходит на этап отгрузки, а вы назначаетесь ответственным литейщиком.

---

## Администратор

Администратор имеет полный доступ ко всем функциям системы.

### Навигация

В шапке приложения доступны пункты:

- **Дашборд** — обзор всех заказов (карточки).
- **Новый заказ** — создание заказа от имени клиента.
- **Заказы** — табличное представление всех заказов с фильтрами.
- **Пользователи** — управление пользователями и ролями.

### Управление пользователями

#### Создание пользователя

1. Перейдите в **«Пользователи»**.
2. Заполните форму «Добавить пользователя»:
   - **Имя** — ФИО сотрудника.
   - **Email** — адрес электронной почты.
   - **Пароль** — минимум 6 символов.
   - **Роль** — выберите: Оператор, Литейщик, Администратор или Клиент.
3. Нажмите **«Создать пользователя»**.

#### Изменение роли

В списке пользователей нажмите на текущую роль и выберите новую из выпадающего списка.

### Управление заказами (таблица)

Страница **«Заказы»** отображает все заказы в табличном виде с колонками:

| Колонка | Описание |
|---------|----------|
| **ID** | Первые 8 символов идентификатора |
| **Создан** | Дата создания (ДД.ММ.ГГ) |
| **Изменён** | Дата последнего изменения |
| **Вид** | Вид работ: **Р** (только Рост) или **Л** (Рост+Литьё) |
| **Клиент** | ФИО клиента |
| **Название** | Название заказа |
| **Папка** | Папка в Галерее |
| **Тариф** | Тариф заказа |
| **Статус** | Текущий этап |

Действия для каждого заказа:
- 👁 **Просмотр** — открыть детали заказа.
- ✏️ **Изменить** — перейти к редактированию.
- 🗑 **Удалить** — удалить заказ (с подтверждением).

Заказы можно фильтровать по этапам с помощью вкладок в верхней части страницы.

### Удаление заказа

Администратор может удалить заказ:
- На странице деталей заказа — кнопка **«Удалить»**.
- В таблице заказов — иконка 🗑 с диалогом подтверждения.

### Перевод статуса

Администратор может перевести любой заказ на следующий этап (кроме уже завершённых).

## Жизненный цикл заказа

Заказ проходит следующие этапы:

```
Черновик → Новый → В росте → В литье → В отгрузке → Завершён
```

| Этап | Кто переводит | Описание |
|------|---------------|----------|
| **Черновик** → **Новый** | Клиент | Отправка заказа в работу |
| **Новый** → **В росте** | Оператор / Админ | Начало выращивания; назначается оператор |
| **В росте** → **В литье** | Оператор / Админ | Передача на литьё |
| **В литье** → **В отгрузке** | Литейщик / Админ | Завершение литья; назначается литейщик |
| **В отгрузке** → **Завершён** | Админ | Заказ выполнен |

## Работа с STL-файлами

### Загрузка файлов

- STL-файлы загружаются при создании заказа.
- Файлы хранятся в защищённом хранилище.
- Имена файлов с кириллицей автоматически транслитерируются для совместимости.

### Скачивание файлов

На странице заказа напротив каждого файла есть кнопка **⬇ Скачать**.

### 3D-превью

1. На странице заказа нажмите **👁 Просмотр** напротив STL-файла.
2. Откроется модальное окно с интерактивной 3D-моделью:
   - **Вращение** — зажмите левую кнопку мыши и двигайте.
   - **Масштаб** — колёсико мыши.
   - **Перемещение** — зажмите правую кнопку мыши.
   - Модель автоматически вращается.
3. Закройте окно крестиком или кликом вне модального окна.
